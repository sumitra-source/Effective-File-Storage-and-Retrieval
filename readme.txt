
To run HuffmanComp and HuffmanDecomp
g++ HuffmanComp.cpp -o HuffmanComp.exe
and then ./HuffmanComp inputfilename.txt

decompress is similar
g++ HuffmanDecomp.cpp -o HuffmanDecomp.exe
./HuffmanDecomp.exe inputfilename-compressed.bin


For running LZW
g++ main.cpp -o main.exe
./main.exe inputfilename
(just input file name without extention)

For EditDistance
g++ EditDistance.cpp -o EditDistance.exe
./EditDistance.exe
